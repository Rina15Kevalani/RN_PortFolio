OTP Verification

The project starts with a function generateOTP that generates a random 4-digit OTP. The sendOTP function simulates sending the OTP, showing an alert with the generated OTP (in a real-world scenario, this would be sent via email or SMS). The verifyOTP function is triggered when the "Verify OTP" button is clicked. It retrieves the entered OTP from the input field and calls the alert with get the OTP sent successfully. It compares the entered OTP with the sent OTP and alerts the user about the verification status.

Frontend: Html,css,javascript
